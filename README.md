# 文件占用查看器

一个 Windows 托盘小工具，用来快速查看资源管理器中选中的文件或文件夹正在被哪些进程占用。

## 使用方法
1. 运行 `FileLockHotkey.exe`。
2. 在 Windows 资源管理器中单击选中文件或文件夹，也可以按住 `Ctrl` 多选。
3. 按快捷键，默认是 `Ctrl+Space`。
4. 在弹窗中查看占用进程、PID、进程类型和程序路径。

托盘图标右键可以手动触发检测、设置快捷键或退出程序。

## 自定义快捷键

1. 右键托盘图标。
2. 点击“设置快捷键”。
3. 在设置窗口中直接按下新的组合键。
4. 点击“保存”。

快捷键会保存到当前用户配置里，下次启动自动沿用。为了避免误触，快捷键至少需要包含 `Ctrl` 或 `Alt`。

## 说明

- 文件查询使用 Windows Restart Manager API，文件夹查询使用系统句柄扫描，不依赖第三方工具。
- 如果某些系统级进程或杀毒软件没有被列出，通常是权限或底层内核句柄限制导致。
- “结束选中进程”会先弹出确认框，请谨慎使用。
